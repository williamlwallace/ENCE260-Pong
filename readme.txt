 // HOW TO MAKE PROGRAM //
 
 1. Download repository from GitLab
 2. Player 1 open folder in terminal and run command 'make program'
 3. Player 2 open folder in different terminal and run same command
 4. Face the UCFUNKITS towards each other so that the IR scanners are facing each other
 4. To start playing player 1 press down navswitch and player 2 press navswitch east
 5. To restart game hold down button by IR scanner and press other button by navswitch
